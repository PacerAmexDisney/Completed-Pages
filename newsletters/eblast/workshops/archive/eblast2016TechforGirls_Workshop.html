Technology for Girls: Digital Sandbox, Learn the Code

Saturday, April 23, 2016 from 10:00 a.m. to 12:30 p.m.

Join us for a day of play in a digital sandbox! Middle school girls with disabilities are invited to join mentors from the University of Minnesota's MnDrive program to see how fun programming can be. Join us to explore and learn about programming and wiring using a SparkFun Digital Sandbox. Learn the ins and outs of programming and help mentors build a robotic hand from 3D printed parts following instructions based on code you write! No experience necessary.Register online

Location: PACER Center

Technology for Girls: STEM Club, Learn to Code

Monday, May 2, 2016 from 5:30 p.m. to 7:00 p.m.

Middle school girls are invited to join members of the Green Girls Robotics Club for this session that is all about coding. From the basics, volunteers will guide you through this hands-on, computer lab session using the open-source resource called Scratch that lets you create stories, games, and animations using code. Register online

Location: PACER Center

Technology for Girls: STEM Club, Web Design on Weebly

Saturday, June 4, 2016 from 9:00 a.m. to 10:30 a.m.

Bring your creativity to PACER and learn how to design a website using Weebly. In this hands-on lab session the Green Girls robotics team will lead middle school girls with disabilities through the basics of web design. Participants will learn how these critical technology skills can be applied in the real world and will create a website using the drag-and-drop website editor Weebly. Beginners welcome. Register online

Location: PACER Center
